## 三、最常用的操作——核验线下支付（每天必做）

顾客转账后，你**必须在这里确认收到款**，商家才会出餐：

```
1. 侧栏点「订单管理」→「线下支付」
2. 找到那笔订单，点「验证支付」按钮
3. 检查顾客上传的转账截图，确认金额和收款账号正确
4. 点「确认已收款」→ 订单自动推进，商家开始出餐
```

⚠️ **没有核验 = 商家不出餐 = 顾客投诉**

## 五、紧急操作速查

**立即关闭某个商家（封号）**
→ 商家管理 → 商家列表 → 找到该商家 → 右侧开关拨为"关闭"

**给顾客手动退款/补偿**
→ 用户管理 → 用户 → 找到该顾客 → 钱包 → 手动添加余额

**解除风控拦截，让某笔订单放行**
→ 风控中心 → 审核队列 → 找到该订单 → 点"放行"

**查看平台今天总收入**
→ 仪表盘顶部卡片（"平台 GMV 大盘"）
